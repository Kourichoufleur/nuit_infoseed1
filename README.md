============= LISEZ MOI ================
Ce script est un document python executable avec un environnement virtuel venv. Pour pouvoir vous assurer de lancer le fichier de départ (main.py) sans avoir à importer de module,
effectuez la commande
  > .venv\Scripts\activate  (en étant sur le terminal du dossier du repo)
Une fois connecté à l'environnement ((.venv) écrit à gauche de la ligne de commande) lancé simplement
  > python3 main.py

Si vous n'y arrivez pas, vous pouvez également
  > pip install pillow, screeninfo
et lancez sur votre propre version de python (nous avons utilisé Python 3.12)
>
